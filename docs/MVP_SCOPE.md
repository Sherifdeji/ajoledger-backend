# AjoLedger MVP Success Criteria & Scope

## Primary Objective
The MVP will be considered successful if a savings group can execute a full lifecycle without relying on manual spreadsheets or external record-keeping.

## Feature Checklist
- [ ] **Community Creation:** Coordinators can create a savings group.
- [ ] **Invitations:** Coordinators can generate invite codes and invite members.
- [ ] **Cycle Configuration:** Coordinators can configure a savings cycle.
- [ ] **Inflows:** Contributors can submit contributions seamlessly.
- [ ] **Reconciliation:** The system automatically tracks payments via Nomba integration.
- [ ] **Transparency:** Maintain a transparent, immutable ledger of all transactions.
- [ ] **Dashboards:** Coordinators and Contributors can monitor progress and payout positions.
- [ ] **Disbursements:** The system can successfully process and complete payouts to the active beneficiary.

## Out of Scope (Do Not Build for MVP)
- AI-powered financial insights
- Community lending / Cooperative credit scoring
- Multi-community management
